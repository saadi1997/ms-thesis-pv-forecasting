# Data

The raw DKASC dataset is not included in this repository due to file-size considerations.

The dataset used in this thesis was downloaded from the Desert Knowledge Australia Solar Centre (DKASC), Alice Springs facility.

Selected system:

- Site: Site 13 / DKA-M6-B-Phase
- Technology: Trina monocrystalline silicon PV
- Array type: Fixed
- Approximate capacity: 5.3 kW
- Installation year: 2009
